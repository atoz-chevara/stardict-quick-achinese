Stardict Quick package for Achinese dictionary of foreign words

This is autogenerated dictionary for StarDict.

Data were downloaded from following website:

https://l10n-indonesian.alioth.debian.org/pub/stardict/quick

The original source is available under GPL-3.

Install dictionary by copying dictionary files to dic/ folder in StarDict. On Linux it is usually /usr/share/stardict/dic/ on Windows C:\Program files\stardict\dic.
